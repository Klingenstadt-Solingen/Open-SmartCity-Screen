# Solingen Digital Signage

This is an electron app, running a digital signage react application.

It runs with

- `NodeJS v18.17`,
- `NextJS v12.3`,
- `yarn v1.22`

## Setup

### Install Dependencies

run `yarn install` to install third party dependencies

### Use it

```
# development mode
$ yarn dev

# production build
$ yarn build
```
